This is an AI chatbot, trained from a certain bank account creation instructions created for Intellihack.


INSTRUCTION TO RUN THE CHATBOT
------------------------------

- Run this from google collab itself using an engine with a GPU.
- Generate a google api key using this link "https://aistudio.google.com/app/apikey" and enter it in the api space.
- Run all the code snippets from the start, before the running the 5th snippet, make sure to put the pdf file containing the information into the data folder. 
- After all of the above snippets have been run, run the last snippet.
- Use the last code snippet to operate the chatbot.
